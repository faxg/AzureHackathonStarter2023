# Setup your Developer Environment
Here you'll find all kinds of information on how to setup and configure an efficient development environment.


### Azure CLI
The Azure CLI tool helps with local development and deployment to Azure Cloud
- [Install Azure CLI](https://learn.microsoft.com/en-us/cli/azure/install-azure-cli)

### Azure VS Code extensions
VS Code has great extensions for all kinds of Azure services. 
- [Azure Extensions for VS Code](https://code.visualstudio.com/docs/azure/extensions)

### Github Codespaces
Fully managed dev-container. Run in the cloud, directly from your Github repository.
- [Github Codespaces](https://github.com/features/codespaces)
