# Manage your Data
Manage your data using cloud storage and managed databases (DBaaS).

Overviews:
- Storage: https://learn.microsoft.com/en-us/azure/storage/common/storage-account-overview
- Database services: https://docs.microsoft.com/en-us/azure/?product=databases

### Azure Blob storage
Store large amounts of unstructured data (images, documents, log files etc...)
  - Docs: https://docs.microsoft.com/en-us/azure/storage/blobs/

  - Quickstarts
    - [Python](https://learn.microsoft.com/en-us/azure/storage/blobs/storage-quickstart-blobs-python?tabs=managed-identity%2Croles-azure-portal%2Csign-in-azure-cli)
    - [JavaScript](https://learn.microsoft.com/en-us/azure/storage/blobs/storage-quickstart-blobs-nodejs?tabs=managed-identity%2Croles-azure-portal%2Csign-in-azure-cli)
    - [.NET](https://learn.microsoft.com/en-us/azure/storage/blobs/storage-quickstart-blobs-dotnet?tabs=visual-studio%2Cmanaged-identity%2Croles-azure-portal%2Csign-in-azure-cli%2Cidentity-visual-studio)
    - [Java](https://learn.microsoft.com/en-us/azure/storage/blobs/storage-quickstart-blobs-java?tabs=powershell%2Cmanaged-identity%2Croles-azure-portal%2Csign-in-azure-cli)

### Azure Cosmos DB 
Fully managed, distributed NoSQL & relational database for modern app development. 
  -  Docs: https://docs.microsoft.com/en-us/azure/cosmos-db/

### Azure Database for PostgreSQL
Relational database service based on the open-source Postgres database engine
  - Docs: https://docs.microsoft.com/en-us/azure/postgresql/
