# Azure Passes

You want to use Azure for your project?
We have voucher codes for you:

-How to get Free Azure Credits...
- Go to our Content page: https://aka.ms/AAmopw4
<img src="/img/HackZurich2023_15092023_142724.png" />
- Register with your e-mail for the <b>Microsoft.Source</b> newsletter
- Click the button to get to the Azure Check-In
- You'll need to auth with your Github account
- You'll see a "Boarding Pass" with a voucher code. Copy to clipboard and click the link that says "Redeem".
- Follow the process
